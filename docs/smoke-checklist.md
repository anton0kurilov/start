# Smoke Checklist

Базовый чек-лист для ручной проверки ключевых пользовательских сценариев после изменений.

## Инициализация и базовое состояние

- Открыть приложение с чистым `localStorage`: видно пустое состояние колонок и настроек.
- Кнопка обновления доступна, статус ошибок скрыт.
- Панель настроек открывается и закрывается (кнопка, backdrop, `Esc`).

## Папки и подписки

- Создать папку: папка появляется в списке и в селекте формы подписки.
- Добавить подписку в папку: подписка появляется в списке папки.
- Удалить подписку: запись удаляется, интерфейс остается консистентным.
- Удалить папку: удаляются и папка, и ее подписки.

## Обновление лент

- Нажать `Обновить`: появляется loading-статус, кнопка временно disabled.
- По завершении обновления кнопка снова enabled.
- При повторных быстрых кликах в `Обновить` не запускается несколько refresh одновременно.
- При отсутствии подписок показывается статус `Добавьте потоки для обновления`.

## Работа с карточками и read-статусом

- Клик по карточке новости отмечает ее как прочитанную.
- Кнопка `Отметить как прочитанное` в колонке переключает read/unread для карточек.
- Клик по заголовку колонки прокручивает список вверх.

## Авто-отметка при скролле

- Включить `Отмечать как прочитанное при прокрутке`.
- Прокрутить колонку вниз и вверх: скрытые сверху карточки отмечаются как прочитанные.
- Выключить опцию: автопометка больше не срабатывает.

## Импорт, экспорт, сброс

- Экспорт JSON скачивает файл и показывает статус `Экспортировано в JSON`.
- Импорт валидного JSON заменяет состояние и обновляет UI.
- Импорт невалидного JSON показывает ошибку без поломки текущего состояния.
- `Сбросить настройки` очищает папки и подписки, интерфейс возвращается к начальному состоянию.
